Interface: Queue

  1. Constructor Function
    • Storage //use object as a storage
  2. Methods
    • enqueue(val) //adds value to the back, returns size
    • dequeue() //removes value from the front, returns value
    • peek() //same as dequeue, but does not remove the front element only returns it
    • size() //returns value of queue as an integer
